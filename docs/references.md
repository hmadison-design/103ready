# 103ready.com — Reference Library

All reference materials are stored in **Google Drive** under the folder **103ready Reference Library** to keep the local SSD and GitHub repo lean.

When working on a scenario in Cowork, Claude can search Google Drive to find documents. For PDFs that need processing (chart extraction, etc.), temporarily place the file in the mounted CYOA folder.

## Google Drive Folder Structure

```
103ready Reference Library/
├── FAA Pubs/                    ← All FAA publications (69 files, ~1.4 GB)
├── Human Factors Research/      ← Academic papers and reports (13 files, ~29 MB)
├── Sectional Charts & Plates/   ← VFR sectionals, TACs, etc.
│   ├── Dallas-Ft_Worth.pdf
│   └── (all VFR raster charts — see below)
└── (future: IFR charts, approach plates, etc.)
```

## FAA VFR Raster Charts

Complete set downloaded from: https://www.faa.gov/air_traffic/flight_info/aeronav/digital_products/vfr/

Includes all VFR Sectional Charts, Terminal Area Charts (TACs), and VFR Flyway Planning Charts. These are stored in the Sectional Charts & Plates folder. Charts expire on a 56-day cycle but are retained for scenario development reference.

**IMPORTANT:** All charts used in scenarios must include the following disclaimer:

> *This chart is provided for training and educational purposes only. It is not intended for navigation and may not reflect current aeronautical information. Always use current, officially published FAA charts and publications for flight planning and navigation. Chart data is sourced from FAA VFR Raster Charts and is subject to a 56-day amendment cycle.*
